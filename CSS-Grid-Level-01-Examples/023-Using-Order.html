Using Order

Grid suporta a propriedade order também encontrada no Flexbox. Se você estiver posicionando explicitamente Itens da Grade, a propriedade order afetará a ordem da pintura e, portanto, a ordem na qual os itens se acumulam onde nenhum índice z foi aplicado.

Se estiver usando o posicionamento automático, a propriedade do pedido afetará como os itens são colocados na grade. Neste exemplo, todas as caixas receberam uma ordem de 1 e, portanto, continuam sendo posicionadas na ordem DOM. No entanto, a caixa1 tem uma ordem de 3 e a caixa2, uma ordem de 2. Essas caixas têm um valor de ordem mais alto, portanto, são posicionadas após todas as caixas com um valor de ordem 1.
