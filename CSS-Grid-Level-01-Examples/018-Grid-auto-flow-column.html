Grid auto-flow column

O comportamento padrão do Grid Auto Flow é o layout dos elementos por linha, trabalhando ao longo da linha até que não haja mais slots e passando para a próxima linha. Se uma linha não for declarada, será criada uma faixa de grade implícita para reter os itens.

Você pode alterar esse comportamento usando a propriedade grid-auto-flow. O valor padrão é linha, mas você também pode especificar a coluna. Os elementos serão dispostos coluna por coluna, adicionando novas colunas, se necessário.
