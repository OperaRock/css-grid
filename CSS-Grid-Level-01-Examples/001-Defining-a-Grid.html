Defining a Grid

Para definir uma grade, use display: grid ou display: inline-grid no elemento pai. Você pode criar uma grade usando as propriedades grade-modelo-colunas e grade-modelo-linhas.

Estou usando a propriedade gap da grade para criar um gap entre minhas colunas e linhas de 10 px. Essa propriedade é uma abreviação para gap de coluna da grade e gap de linha da grade, para que você possa definir esses valores individualmente.

Todos os filhos diretos do pai agora se tornam itens de grade e o algoritmo de posicionamento automático os expõe, um em cada célula da grade. Criando linhas extras, conforme necessário.
