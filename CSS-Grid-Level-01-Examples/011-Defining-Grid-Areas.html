Defining Grid Areas

Podemos criar áreas nomeadas na grade para inserir conteúdo. Para fazer isso, primeiro atribuímos elementos em nossa marcação a uma Grid Area, usando a propriedade grid-area-property.

Podemos então usar a propriedade grid-template-areas para descrever onde esses elementos devem estar na grade.

Repetir o nome de uma área indica que a área abrange a trilha da grade. Usando um '.' ou uma sequência como '....' indica uma faixa vazia. Como a primeira célula nesta grade.
