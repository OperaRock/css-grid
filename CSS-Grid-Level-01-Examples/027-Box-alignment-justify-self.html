Box alignment justify-self

Eu usei uma imagem de plano de fundo da grade para mostrar a grade neste exemplo. Eu defini quatro áreas de grade de tamanho igual à esquerda, que cobrem uma área quadrada dentro de três linhas de linha e coluna.

Estou usando a propriedade justificar-se em itens de grade individuais para demonstrar os diferentes valores.

This is box A.

justify-self: stretch

This is box B.

justify-self: end

This is box C.

justify-self: start

This is box D.

justify-self: center

Cada uma das caixas à esquerda possui uma área de grade de 3 colunas e 3 linhas (contamos o gutter coluna / linha). Portanto, cada um cobre a mesma área de tamanho da caixa A.

A propriedade justify-self é usada para justificar o conteúdo dentro da área da grade.
