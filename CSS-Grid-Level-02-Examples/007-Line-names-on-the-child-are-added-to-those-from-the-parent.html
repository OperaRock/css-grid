Line names on the child are added to those from the parent

Se você definir nomes de linhas na sub-grade, eles serão adicionados a quaisquer linhas definidas no pai.

Os nomes de linhas são adicionados a uma sub-grade, listando os nomes após a palavra-chave sub-grade, por exemplo template-columns: subgrid [sub-1] [sub-2] [sub-3] [sub-4];
