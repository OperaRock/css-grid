As many as will fit with some items spanning two rows

Este exemplo mostra um padrão de duas colunas, com cabeçalho e rodapé. O painel central contém um número de caixas. Queremos ter o maior número possível em cada linha, com as caixas alinhadas como linhas e colunas.

Alguns de nossos itens abrangem duas linhas.

Essa técnica significa que não precisamos usar consultas de mídia, pois uma grade de layout flexível faz o trabalho para nós.

O fallback usa o Flexbox, precisamos restringir os itens no layout flex para obter a aparência do alinhamento bidimensional. Não temos os itens altos no layout flexível.

Estou usando minmax em dois lugares na minha definição de grade. Para criar a lista de colunas de faixas com no mínimo 200 pixels e no máximo 1fr. Além disso, ao definir linhas, quero que minhas linhas tenham no mínimo 100 pixels de altura, mas se expandam para a altura do que for colocado nelas. Portanto, as linhas têm um máximo de automático.

Técnicas usadas para criar esse layout

  • Defining a grid
  • The fr unit
  • Feature Queries
