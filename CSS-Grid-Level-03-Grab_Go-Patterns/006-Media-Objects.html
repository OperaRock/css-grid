Um conjunto inteiro de exemplos de objetos de mídia com um fallback flutuante. Esses objetos podem ser invertidos, aninhados e, em seguida, podem ter uma coluna fixa para a imagem ou uma coluna flexível.

Este exemplo cria o layout usando grid-template-areas

Técnicas usadas para criar esse layout

  • Grid Template Areas
  • Nested Grids
  • Feature Queries

Dicas e técnicas para layouts de fallback podem ser encontradas em Grid Fallbacks and Overrides Cheatsheet.
